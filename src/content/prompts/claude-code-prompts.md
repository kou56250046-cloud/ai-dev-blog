## コードレビュー

```
このコードをレビューして。特に以下の観点で確認してください：
- セキュリティの問題
- パフォーマンスの改善余地
- 可読性・保守性
- TypeScriptの型の適切さ
```

---

## バグ修正

```
以下のエラーが発生しています。原因を特定して修正してください。

エラー内容：
[エラーメッセージをここに貼り付け]

期待する動作：
[正しい動作の説明]
```

---

## リファクタリング

```
このコードをリファクタリングしてください。
- 重複を排除する
- 関数を適切な単位に分割する
- 変数名をより明確にする
- コメントは日本語で追加する
```

---

## 新機能実装

```
以下の仕様で機能を実装してください：

機能名：
概要：
入力：
出力：
制約条件：
- TypeScriptで書く
- 既存のコードスタイルに合わせる
- エラーハンドリングを含める
```

## ドキュメント生成

```
このコードのドキュメントを日本語で書いてください。
- 概要
- 各関数の説明（引数・戻り値）
- 使用例
- 注意事項
```
